# Site Web Mosquée Bleue - CIMG Villefranche-sur-Saône

Site web officiel de la Mosquée Bleue de Villefranche-sur-Saône.

## Démarrage rapide

Double-cliquez sur `start-server.bat` pour lancer le serveur.

Le site sera accessible sur : http://localhost:3000

## Pages disponibles

- Accueil : http://localhost:3000
- Nos Pôles : http://localhost:3000/poles
- Faire un don : http://localhost:3000/don
- Contact : http://localhost:3000/contact
- Mentions légales : http://localhost:3000/mentions-legales
- Politique de confidentialité : http://localhost:3000/confidentialite
- Connexion admin : http://localhost:3000/admin/login

## Configuration

1. Installer Node.js (v18+)
2. Installer les dépendances : `npm install`
3. Configurer le fichier `.env` (copier `.env.example`)
4. Lancer : double-clic sur `start-server.bat`

## Technologies

- Node.js + Express
- EJS (templates)
- CSS3 + JavaScript vanilla
- Mawaqit API (horaires de prière)
